### Kết quả đạt được tuần 4:

* Lab13 - Củng cố kỹ năng AWS Backup: backup plan, kiểm thử restore và thông báo SNS.
* Lab14 - Di chuyển một máy ảo giữa môi trường on-premises và Amazon EC2 bằng VM Import/Export.
* Lab15 - Đóng gói và triển khai ứng dụng bằng Docker và Docker Compose trên EC2, push image lên ECR.
* Lab16 - Triển khai ứng dụng container trên Amazon ECS với Cloud Map service discovery và Application Load Balancer.
* Lab17 - Xây dựng pipeline CI/CD để tự động hoá việc deploy container lên ECS.

**Dịch vụ AWS đã học trong tuần:** AWS Backup, Amazon EC2, Amazon S3, VM Import/Export, Docker, Amazon ECR, Amazon ECS, AWS Cloud Map, Application Load Balancer, AWS CodeDeploy, AWS IAM, Amazon CloudWatch Container Insights.
