This worklog covers my entire internship at the **Workforce Bootcamp - First Cloud AI Journey** program run by AWS Vietnam, spanning **15 weeks** (April 17, 2026 to July 30, 2026, about 3.5 months). The first 5 weeks were a self-paced AWS Study Group phase covering foundational services; the remaining 7 weeks were spent building the capstone project **PeriodIQ** with a 5-person team, where I owned the **CI/CD & Monitoring** track (AWS CodePipeline, CodeBuild, CloudFormation/SAM, CloudWatch).

**Week 1:** [Getting familiar with AWS: IAM, VPC, EC2](1.1-week1/)

**Week 2:** [Practicing EC2, Amazon RDS, EC2 Auto Scaling, AWS Budgets, CloudWatch](1.2-week2/)

**Week 3:** [Hybrid DNS with Route 53 Resolver, AWS CLI, AWS Backup](1.3-week3/)

**Week 4:** [Deploying Docker, Amazon ECS, and CI/CD with ECS](1.4-week4/)

**Week 5:** [Security Hub, VPC Peering, Transit Gateway, CodePipeline; team kickoff for the capstone project](1.5-week5/)

**Week 6:** [Team kickoff meeting, dividing AWS ownership for the PeriodIQ project](1.6-week6/)

**Week 7:** [Starting PeriodIQ implementation in the CI/CD & Monitoring role](1.7-week7/)

**Week 8:** [Designing the AWS architecture for the CI/CD and monitoring layer](1.8-week8/)

**Week 9:** [Revising the architecture based on team feedback, starting CI/CD backend implementation](1.9-week9/)

**Week 10:** [Building the initial CI/CD and Infrastructure-as-Code foundation (SAM, buildspecs)](1.10-week10/)

**Week 11:** [Stabilizing the CodePipeline/CodeBuild pipeline, upgrading to .NET 10, monitoring dashboard](1.11-week11/)

**Week 12:** [Finishing integration, production deployment, and wrap-up handover](1.12-week12/)
