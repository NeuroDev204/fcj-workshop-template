### Kết quả đạt được tuần 5:

* Lab18 - Bật AWS Security Hub và tìm hiểu cách tích hợp với GuardDuty, Inspector, Macie để giám sát bảo mật và compliance tập trung.
* Lab19 - Kết nối riêng tư hai VPC bằng VPC Peering, cấu hình Network ACL, Security Group và DNS resolution xuyên VPC.
* Lab20 - Tập trung hoá kết nối nhiều VPC bằng AWS Transit Gateway.
* Lab21 - Chưa hoàn thành: trang lab hiện không truy cập được trên AWS Study Group.
* Lab22 - Tự động hoá lịch bật/tắt EC2 bằng AWS Lambda và thông báo Slack để tối ưu chi phí.
* Lab23 - Xây dựng pipeline CI/CD bằng AWS CodePipeline để deploy ứng dụng lên EC2.
* Lab24 - Cấu hình AWS Storage Gateway để chia sẻ file từ cloud xuống hệ thống on-premises.
* Họp team để lên ý tưởng cho project cuối cùng.
* Tham gia workshop do AWS tổ chức vào ngày 23/05/2026.

**Dịch vụ AWS đã học trong tuần:** AWS Security Hub, Amazon GuardDuty, Amazon Inspector, Amazon Macie, Amazon VPC (Peering, NACL), AWS Transit Gateway, AWS Lambda, AWS IAM, Amazon S3, AWS CodeCommit/CodeBuild/CodeDeploy/CodePipeline, AWS KMS, AWS Storage Gateway.

**Kỹ năng đạt được:** Quản lý tư thế bảo mật cloud (security posture); networking nâng cao (VPC Peering, Transit Gateway); tự động hoá tối ưu chi phí serverless; pipeline CI/CD; lưu trữ hybrid; làm việc nhóm lên ý tưởng project.
