### Kết quả đạt được tuần 2:

* Lab04 - Thực hành khởi chạy và quản lý EC2 instance trên Windows và Linux, triển khai app quản lý người dùng Node.js với Security Group, EBS snapshot và custom AMI.
* Lab05 - Học cách triển khai database Amazon RDS với VPC isolation, encryption, backup/restore và Multi-AZ.
* Lab06 - Xây dựng kiến trúc ứng dụng có khả năng mở rộng bằng EC2 Auto Scaling Group, Elastic Load Balancing và Launch Template.
* Lab07 - Học cách quản lý chi phí AWS bằng Cost, Usage, RI và Savings Plans Budget.
* Lab08 - Thực hành giám sát hệ thống bằng Amazon CloudWatch metrics, Logs Insights, Alarms và Dashboards.
* Lab09 - Tìm hiểu về AWS Support Plans và cách tạo, quản lý support request.

**Dịch vụ AWS đã học trong tuần:** Amazon EC2, Amazon VPC, Security Groups, EBS, AMI, AWS IAM, Amazon RDS, EC2 Auto Scaling, Elastic Load Balancing, AWS Budgets, Amazon CloudWatch, AWS Support.
